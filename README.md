# Snowflake Hotel Booking Analysis Project

# Project Overview
This project demonstrates a hotel booking analytics system using Snowflake Data Warehouse.

# Technologies Used
- Snowflake
- SQL
- CSV
- GitHub

# Project Structure
- SQL scripts for database and tables
- Sample CSV data
- Analytical queries

# Key Analysis
- Total bookings per hotel
- Revenue generated per hotel
- Customer booking behavior

# How to Run
1. Create database and schema in Snowflake
2. Run table creation SQL
3. Load CSV data using Snowflake COPY command
4. Execute analysis queries

#Use Case
Useful for hotel chains to analyze bookings and revenue trends.
